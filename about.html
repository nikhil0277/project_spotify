About the Spotify Clone :-

Overview:

This Spotify clone is a simplified web application that mimics the basic functionalities of the popular music streaming service, Spotify. Built with HTML, CSS, and JavaScript, this project serves as an excellent learning tool for understanding web development fundamentals, including layout design, responsive design, and basic JavaScript interactions.

Features :-

1. Music Player Interface:

Displays a list of songs with their respective album covers and names. Allows users to play, pause, and navigate through songs using play, pause, previous, and next buttons. Shows the progress of the current song with a progress bar.

2. Responsive Design:

Ensures the interface looks good on various devices, including desktops, tablets, and mobile phones. Uses media queries to adjust the layout for different screen sizes.

3. Visual Feedback:

Shows a playing GIF when a song is playing. Updates the play/pause icons to reflect the current state of the player.

4. Favicon:

Includes a favicon for a professional touch.

Technical Details:

1.HTML: 
        Defines the structure of the web application, including navigation, song list, and controls.
    
2.CSS:
        Styles the application, ensuring a visually appealing and consistent look. Includes responsive design with media queries for different screen sizes.

3.JavaScript:
        
        Handles the functionality of the music player, including play/pause actions, song navigation, progress bar updates, and volume control.
